1、安装Ubuntu

（1）选择创建新的虚拟机

（2）推荐选择典型，那么如果你没有特殊要默认即可

（3）这里有三种安装方式，一般采用下两种，注意如果是工作的话需要VMware Tools的话不要选择简易安装，选择第三种稍后安装操作系统。



###（4）选择Linux，版本的话做嵌入式的选择32位比较好，不然可能会滋生很多环境问题

（5）输入虚拟机名称及安装位置，然后一路默认直至点击完成



（6）在你创建的虚拟机上右键选择设置，找到CD那一项，选择你下载好的镜像文件，导入镜像文件，然后点击完成，接着只需要启动虚拟机，就会进入安装界面，安装时，选择简体中文， 语言选择汉字，其他全部默认即可，然后慢慢等待安装完成后重启虚拟机即可（虚拟机可能也需要关闭再重新打开，如果Ubuntu一直无法进入的话）。



2、 VMware Tools安装

基本上用虚拟机工作的话这个工具的重要性不言而喻，但是在使用Ubuntu时安装总是出问题，下面我简要总结一下整个安装过程：

（1）鼠标右键虚拟机，选择设置，和上面安装系统是一样的操作，但是这时候你会发现使用ISO映像文件那一项已经变了。（安装完成Ubuntu后进入Ubuntu点击虚拟机下方提示的安装VMtools选项会出现一个光盘）


（2）然后还是右键虚拟机，但是这次是选择安装VMware Tools，这个过程是需要网络的，虚拟机默认NAT模式联网，不需要别人远程连接你的虚拟机的话这里不需要改，如果要的话可以看我的另一个总结：如何让别人远程连接你的VMware虚拟机，或者搜索也可以。


如果没有安装成功，可以点击虚拟机提示的帮助按钮，会打开一个网页，上面有很详细的解决过程。

（2）VMware Tools安装成功后会弹出（如果点击光盘进入后没有内容的话再重启一下Ubuntu）：

（3）这里我们将其中的.tar.gz包拷贝到/tmp/目录下

cp VMwareTools-10.1.6-5214329.tar.gz /tmp/

（4）进入tmp目录下并解压该压缩包

cd /tmp/

tar zxvf VMwareTools-10.1.6-5214329.tar.gz

（5）进入解压后的文件然后进行安装（除了第一个选项输入y其它选项一路enter即可）

cd vmware-tools-distrib/

sudo ./vmware-install.pl

（6）重启

reboot

最后，查看一下拖放和复制黏贴是否启用。
